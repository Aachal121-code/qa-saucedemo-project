**ID:** TC-010

**Title:** Verify checkout flow completes successfully  

**Preconditions:**  
- User is logged in successfully  
- At least one item is added to cart  
- Cart page is accessible  

**Steps:**  
1. Navigate to the **Cart** page  
2. Click on the **Checkout** button  
3. Enter valid **First Name**, **Last Name**, and **Zip Code**  
4. Click **Continue**  
5. Review the order summary  
6. Click the **Finish** button  

**Expected Result:**  
- User redirects to the **Order Completion** page  
- Shows message: **"THANK YOU FOR YOUR ORDER"**  

**Actual Result:**  
- Checkout completes successfully  
- Order confirmation page displays correctly  

**Status:**  Pass
