# TC-004 - Product Listing visible    

**ID:** TC-004  

**Title:** Products Name, Price and Image visible  

**Precondition:**  
- User account exists and successfully sign in  
- Application server is up and running.  

**Steps:**  
1. Open "https://www.saucedemo.com"  
2. Sign in with valid credentials  
3. Redirected to the `inventory` page  

**Expected Result:**  
- User is redirected to inventory page  
- Products are visible with `Name`, `Price`, and `Image`  

**Actual Result:**  
- Inventory page loads  
- Products are visible with `Name`, `Price`, and `Image`  

**Status:** Pass
